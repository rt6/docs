## Express Restful API Backend

```
$ npm install express-generator -g
$ expres my-new-app
$ cd my-new-app
$ npm i
```

The above will create the below structure:
```
├── app.js
├── bin
│   └── www
├── package.json
├── public
│   ├── images
│   ├── javascripts
│   └── stylesheets
│       └── style.css
├── routes
│   ├── index.js
│   └── users.js
└── views
    ├── error.jade
    ├── index.jade
    └── layout.jade
```
